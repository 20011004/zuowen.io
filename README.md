# 形态的变形，人心的变形，世界的变形
<p>
<font size="10" face="Times">
  高度决定视野，视野决定格局。鲲鹏不知飞鸟的落寞，飞鸟不知寒蝉的渺小。但，假如有一天寒蝉翱翔天际，鲲鹏却蜷缩于泥土，那么一切会如何呢？形态的变形并可怕，可怕的是随之而来的人心的变形、世界的变形。
  万千世界，人们为了追逐名利、金钱、地位而一天天的抛弃自己的内心，扑向花红酒绿的浮华与虚伪，做着一次又一次痛苦并欢悦的变形。那么，所谓的人性还剩多少呢？而随之而来的残酷社会驱动力又会膨胀多少呢？
  形态的变形，往往也意味着人心的变形。就像我之前看的一部动画里所描述的一样，一个平凡的小职工一夜醒来却得到了超人的能力，然后，他报复曾欺侮过他的人，折磨弱者，每天过着疯狂的生活。其实心理学中也有这个概念，这个概念被称为路西法效应，它指出几乎每个人心中都有一个恶魔，只是平常时被社会的条例、以及没有超越规则的能力所压制。就像弗洛伊德指出的自我、本我、与超我一样。但，一旦自己本身发生了极大的变化，就像上文所提到的一夜从一个普通人变成了超人，拥有了可以藐视规则而不用担心惩罚的能力后，那个恶魔也就是本我便释放了，就像你被上司因一件小事而借题发挥对你大吼大叫时，想必很多人都在心里想象过自己拥有强大的power，也就是有权又要能力后将上司百倍报复，虽说这仅仅是种阿Q精神，但万一你有一天真的又有了这种power呢？凯撒、希特勒、《守财奴》中的主人公都将此发挥到了一种极致。而在此我所指的形态不仅仅指的是外貌的变化，更多的则是外在power的变化，由此可见，在未有与强大心态之前形态的变化引起更多的则是人心的变化。
  若鲲鹏化为寒蝉呢？或许随之而来的是一种处境的变化吧。卡夫卡《变形记》的主人公格里高利原本就是一个原本为全家赚钱被全家人爱戴的小职员，直到有一天它醒来发现自己竟变成了一只巨大的甲虫，然而讽刺的是他想到的第一件事并非惊讶，而是考虑自己这个样子该怎么去工作。呵，好一个柏拉图的理想国度啊。不久后他的家人便开始嫌弃他，嫌弃他是个没用的拖油瓶，直至最后被他所最爱的妹妹连同家人赶出家门，死在大街上告结。更为更讽刺的是全书的结尾，因家里没有格里高利这个拖油瓶全家的生活又开始变得蒸蒸日上了。《高老头》中高老头从一个坐拥百万资产被女儿左亲右抱的富人变成了一个一穷二白孤独死去的老者不亦讲述了这个道理吗？这一切无不展现着因自身形态改变而招致处境改变的道理，而所谓的处境不也就是社会吗？就像一个商人在自己成功时朋友成群，亲戚成片仿佛整个社会、整个世界都在向它微笑问好，然而当他失败时呢，无论亲人还是朋友都无一人伸手相助，皆冷眼相看。虽残酷，但事实就是如此。每个人都将自己所了解的当做世界本身而唯有变形后才发现，自己所看到的仅仅是世界的一角罢了。
  朝菌不知晦朔，蟪蛄不知春秋。朝菌不会理解蟪蛄的处境，蟪蛄亦不了解朝菌的心态，很多的时候形态的变形会导致世界的改变。如今是一个高速发达的世界商人逐利，本无对错，
就像克莱齐奥所说的一样“在加速奔跑的生命中，一切的善与恶、美与丑都会像路边的树木一样后退、淡远、消失”当今的世界犹如一辆飞驰的电车若贸然冲撞势必会粉身碎骨，所以我们既需要追赶世界的步伐使自己一次次的变形又需要有一颗坚定不被尘世烟火沾染的心态从而应对世界的变形。
